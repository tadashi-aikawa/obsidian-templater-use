# obsidian-templater-use

## For developers

### Required

- [Bun] (v1.0.7 >=)

### Provisioning

```console
bun install
```

### Build

```console
bun build:all
```

#### Artifacts

- `dist/index.js`
- `dist/index.d.ts` (Optional)

### Release

```console
VERSION=20231021.1 bun release
```

[Bun]: https://bun.sh/
